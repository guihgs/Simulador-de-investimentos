# Ciclic Teste Front

## Para instalar as Dependências
npm install

### Para rodar o projeto com hot-reloads
npm serve
ou
yarn serve

### Para compilar o projeto para produção
npm build 
ou
yarn build

### Para rodar testes
npm test
ou
yarn test

## Desenvolvido usando
- Vue.js
- Testes com Jest
- CSS pré-processador SASS
